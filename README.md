## Running Locally

Clone the repository:

```bash
git clone https://github.com/YousefKart/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
